# Issue Tracker: GitHub

Issues and PRDs for this repo live as GitHub issues.

- Use the `gh` CLI for all operations.
- Understand and respect relationships of issues.
- Add the "sub-issues / parent issue" relationship when appropriate. For
  example, an PRD can be a parent issue and specific tasks are sub-issues.
- Add the "blocking / blocked-by" relationship when there is such a
  relationship. Do not try to solve an issue when it's blocked by other open
  issue(s). Tackle blocking issues first.
